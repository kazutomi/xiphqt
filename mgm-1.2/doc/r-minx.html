Resource: minx

applicability

Module class or instance

possible settings

>0 (default varies)

description

Minx sets the minimum allowed size alloted for a given module's widgets in the main MGM window. The main MGM window cannot be resized to a width less than the sum of all modules minx settings (when stacked horizontally) or the maximum of all the module minx settings (when stacked vertically). The geometry resource can be used to force MGM to a smaller size than minx (and miny) would normally allow. This will probably break things. Beware.

Minx may be used to force the minimum size larger, or may be used (along with scalelenadj and scalewidadj) to force the module to a specific absolute size that won't change. See the geometry page for details.

see also

geometry miny scalelenadj scalewidadj
