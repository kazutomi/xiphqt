Module: loadav

Displays the traditional UNIX load averages for the past minute, five minutes and 15 minutes.

Current platforms

Linux

Instances

single only

additional resources

none

Resource defaults

Resource defaults explicitly altered from the global defaults by this module:
  • bar.0.label: Load average
  • bar.1.label: Load (5min)
  • bar.2.label: Load (15min)
  • order: 2
  • scalelenadj: 60
  • scalerefresh: 2000
  • scalewidadj: 220

Possible enhancements

Could (and should) be extended to use uptime and other utils on platforms other than Linux (eg, Solaris *BSD).
